Events

[data-toggle="scrollspy"] click

ScrollSpy 기능
Parameters:
Name Type Description
href ID 해당 ID를 가진 엘리먼트의 최 상단으로 이동
Selector data-avoid-target target 된 스크롤 시 엘리먼트의 높이값을 더함
Examples
// Basic
<a href="#target" data-toggle="scrollspy"> 
  target으로 이동 
</a>
// data-avoid-target 적용
<a href="#target" data-toggle="scrollspy" data-avoid-target=".gnb"> 
  target의 위치와 $('.gnb') 높이를 더한 값으로 이동
</a>
