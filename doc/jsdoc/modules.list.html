label click

- 해당 label 태그에 checked 클래스를 Toggle
- 체크박스 인 경우 for 가 향하는 체크박스에 checked 속성을 True
- 라디오 버튼 인 경우 for 가 향하는 라디오 버튼에 checked 속성을 True
Examples
// 체크박스
<label for='checkbox'>
  <i></i>
</label>
<input type="checkbox" id="checkbox" class="is-custom"/>
// 라디오 버튼
<label for='radio'>
  <i></i>
</label>
<input type="radio" id="radio" name="radio" class="is-custom"/>

[data-toggle="scrollspy"] click

ScrollSpy 기능
Parameters:
Name Type Description
href ID 해당 ID를 가진 엘리먼트의 최 상단으로 이동
Selector data-avoid-target target 된 스크롤 시 엘리먼트의 높이값을 더함
Examples
// Basic
<a href="#target" data-toggle="scrollspy"> 
  target으로 이동 
</a>
// data-avoid-target 적용
<a href="#target" data-toggle="scrollspy" data-avoid-target=".gnb"> 
  target의 위치와 $('.gnb') 높이를 더한 값으로 이동
</a>

ui__scroll-top click

Scroll To Top 이벤트 핸들러
문서의 최 상단으로 올라가게 하는 기능 (700ms)
Example
<a role="button" className="ui__scroll-top"></a>
