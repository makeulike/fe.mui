Class: Sequence

Sequence

new Sequence(width, height, scene, speed, selector, isReplay)

시퀀스 이미지에 대한 하나의 인스턴스를 생성한다.
Parameters:
Name Type Description
width Int 한 컷의 이미지 넓이
height Int 한 컷의 이미지 높이
scene Int 컷 갯수
speed Int ms 단위의 컷 당 변화 시간
selector DOM background-image 로 지정되어 있는 DOM
isReplay Boolean 반복재생 여부

Methods

Parameters:
Name Type Description
type String 한번 재생 시 "once" / 없을 시 빈값
